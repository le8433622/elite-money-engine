# Customer Journey

## Stage 1: Discovery
The user finds the product through content, referrals, or direct outreach.

## Stage 2: Signup
The user creates an account and sees a clear promise: better money tracking and smarter alerts.

## Stage 3: Onboarding
The user adds first transactions, sets budgets, and enables at least one automation rule.

## Stage 4: First value moment
The dashboard shows financial clarity and automation surfaces a useful alert.

## Stage 5: Habit formation
The user returns weekly to track cash flow and monitor budgets.

## Stage 6: Upgrade decision
The user sees value in richer automation, reports, and advanced insights, then upgrades to Pro.

## Critical moments to optimize
- landing page clarity
- signup simplicity
- time to first transaction
- time to first alert
- time to first report
